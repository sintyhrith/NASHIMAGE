Cricket Team Builder (Web Only)

Features:
- 25 players always visible
- Select Top 12 (T20)
- Assign Captain & Vice-Captain
- Drag / Fixed positions on ground
- Mobile-friendly
- Save / Load team (LocalStorage)
- Export team as JSON
- Export ground as PNG image
- Dream11-style UI
- Free hosting ready (GitHub Pages / Netlify)

Deployment:
- GitHub Pages: upload files → Settings → Pages → main branch → live URL
- Netlify: drag folder → done
